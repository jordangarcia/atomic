# The Atomic Editor

Trivializing iframes...

## Usage

#### Installation

```sh
npm install
```

**Starting**

```sh
gulp
```

Will start a webserver for innie/outtie and do a webpack watch on both of the js bundles

(`npm install -g gulp` if you dont have gulp)

`http://localhost:8001/`
